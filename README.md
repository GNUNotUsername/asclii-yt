# ASClii-YT

Play a youtube video in your terminal window.

# Usage
```python asclii-yt.py <url> <width>x<height>```

where:
* ```<url>``` is the URL of the video to play.
* ```<width>``` is the width of the picture to display in the terminal window.
* ```<height>``` is the height of the picture to display in the terminal window.

# Dependencies:
* ffmpeg
* python-vlc
* python-opencv
* python-pytube
* python-pillow
* python-python-ffmpeg
